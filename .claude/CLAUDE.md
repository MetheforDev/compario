# Compario Projesi

Bu proje: Türkiye'nin evrensel karşılaştırma platformu. Araç, teknoloji ürünleri ve daha fazlasını karşılaştırarak kullanıcıların bilinçli kararlar almasını sağlar.

## Tech Stack
- Next.js 14 (App Router)
- Supabase (Database + Auth)
- Turborepo monorepo
- Tailwind CSS
- TypeScript

## Proje Yapısı
- `/apps/web` - Next.js uygulaması
- `/packages/database` - Supabase queries
- `/scripts` - Utility scripts
